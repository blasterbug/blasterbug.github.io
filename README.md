## Usage


### Install

```bash
  $ gem install bundler jekyll
  $ bundle install
  $ bundle exec jekyll serve
```


### Dev mode

```bash
  $ bundle exec jekyll serve
```
Now browse to [localhost:4000](http://localhost:4000).


### Build

```bash
  $ bundle exec jekyll build
```
