## Usage

### Install

```bash
  $ yarn install
```

### Dev mode

```bash
  $ yarn start
```

Now browse to [localhost:4000](http://localhost:4000).

### Build

```bash
  $ yarn build
```

### Deploy to GitHub pages

```bash
  $ yarn deploy
```
